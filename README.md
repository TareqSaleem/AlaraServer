#ALara Alara Lukagro Server


## Setup

### Clone the repository
```
git clone https://github.com/TareqSaleem/AlaraServer.git
```
## install modules
```
cd functions
npm install
npm install express body-parser nodemailer cors --save
```
### Run - Development
```
npm start
```
##Navigate to static production server

http://localhost:4444/
